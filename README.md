# kalendaro-holidays
Выходные дни с 2012 по 2018 гг.

## Формат

Пример данных:
```json
{
  "2012": {
    "1-1": 1,
    "2-1": 0
  }
}
```

Описание:
```
{
  "Год": {
    "День-Месяц": 1, // 1 - выходной день
    "2-1": 0 // 0 - рабочий день (суббота или воскресенье), появляется при переносах
  }
}
```
